#Habitude
This app was created as an entry to TechTogether Boston 2020's hackathon.

This app was created to help users form lasting habits.

##Technologies Used:
- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- MongoDB
- MongoDB Cloud Atlas
- Mongoose
- React.js
